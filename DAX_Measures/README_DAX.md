# DAX Measures Documentation

This folder contains all DAX measures used to build the **Banking Analytics Power BI Dashboard**.

## Dashboard Pages

### Page 1 — Transaction Analysis
- Total Transactions
- Total Transaction Amount
- Debit %
- Credit %

### Page 2 — Customer & Account Insights
- Total Customers
- Total Accounts

These measures were created using Power BI DAX to calculate KPIs and business metrics for interactive reporting.
