# SQL Workflow Documentation

This project simulates a real-world banking database system using Microsoft SQL Server.

---

## Steps to Run SQL Scripts

1. Execute `01_Database_Setup.sql`
2. Run `02_Data_Generation.sql`
3. Execute `03_Master_Data_Insert.sql`
4. Run `04_Data_Cleaning.sql`
5. Execute `05_Data_Modeling_and_Analysis.sql`

---

## Final Dataset Created

**CombinedBankingDataset**

This dataset is used in Power BI for dashboard creation and business analysis.
